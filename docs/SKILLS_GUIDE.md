# Гайд по скиллам для AI CITI

## Когда какой скилл вызывать:

### `/product-manager`
**Когда:** Получил фидбек от пользователя, нужно понять что делать
- Анализ отзывов и жалоб пользователей
- Приоритизация фич (RICE скоринг)
- Написание ТЗ/PRD на фичу
- Customer discovery интервью

**Пример:**
```
Вот фидбек от пользователя карусельного генератора:

"Текст огонь, но я выбирала слово ХОЧУ, а появилось МАГИЯ.
И было бы круто сначала увидеть текст, а потом дизайн —
страшно потратить монеты и увидеть фразу которую нельзя исправить"

Разбери по косякам, дай приоритеты что фиксить.
```

---

### `/marketing-ideas`
**Когда:** Нужны идеи как привлечь пользователей
- Viral loops для Telegram
- Каналы привлечения
- Retention механики
- Реферальные программы

**Пример:**
```
AI генератор каруселей для Instagram (Telegram Mini App).
ЦА: сетевики, МЛМ в России.
Цена: 90 рублей за карусель.
Есть реферальная программа 10%.

Нужны идеи:
1. Viral loops в Telegram
2. Каналы привлечения
3. Как удержать юзеров
```

---

### `/marketing-strategy`
**Когда:** Стратегические маркетинговые решения
- Позиционирование продукта
- Выход на новый рынок
- Анализ конкурентов (battlecards)
- Планирование запуска фичи

**Пример:**
```
Продукт: AI карусели для Instagram
Конкуренты: Canva, ручной дизайн, фрилансеры

Как позиционироваться? В чём наше УТП?
Как объяснить ценность за 3 секунды?
```

---

### `/product-strategist`
**Когда:** Большие стратегические решения
- Квартальные OKR
- Roadmap на 3-6 месяцев
- Монетизация и unit-экономика
- Масштабирование команды

**Пример:**
```
Реферальная программа AI карусельного генератора:

- Цена карусели: $3 (30 нейронов)
- Себестоимость: $1.4 (API)
- Маржа: 47%
- Текущий реферал: 10% от трат партнёра

Вопросы:
1. Какой % оптимальный чтобы и мотивировать и не в минус?
2. Нужен ли бонус за регистрацию?
3. Одноразовый vs рекуррентный бонус?
```

---

### `/copywriting`
**Когда:** Нужны конкретные тексты
- Тексты для лендинга
- CTA кнопки
- Онбординг сообщения
- Описания фич

**Пример:**
```
Перепиши блок "Как это работает" для партнёрки:

Текущее:
1. "Отправь ссылку другу" / "Он регистрируется"
2. "Друг тратит нейроны" / "На любые инструменты"
3. "Ты получаешь 10%" / "От каждой траты"

Нужно: короче, цепляющее, для сетевиков.
```

## Быстрая шпаргалка:

| Задача | Скилл |
|--------|-------|
| Фидбек от юзера | `/product-manager` |
| Как привлечь юзеров | `/marketing-ideas` |
| Позиционирование | `/marketing-strategy` |
| OKR, экономика | `/product-strategist` |
| Написать текст | `/copywriting` |

---

## Комбо-примеры:

**Запуск новой фичи:**
1. `/product-manager` — написать PRD
2. `/copywriting` — тексты для UI
3. `/marketing-ideas` — как продвигать

**Получил жалобу:**
1. `/product-manager` — разобрать и приоритизировать
2. Пофиксить код
3. `/copywriting` — обновить тексты если нужно

**Планирование квартала:**
1. `/product-strategist` — OKR и приоритеты
2. `/marketing-strategy` — GTM план
3. `/marketing-ideas` — конкретные тактики
